# Tunisia Stock Portfolio Screener

A small toolkit for tracking the Tunis Stock Exchange (TUNINDEX) and
checking a recommended stock portfolio against the live market. Built
around real, current data from Tunisian financial sites, not backtested
history.

## What this project does

The tool loads Tunisie Valeurs' 2025 model portfolio, a list of 11 core
stocks and 5 dividend-yield stocks, and checks each name against the live
list of companies on the Tunis Stock Exchange. It reports which
recommended names show up in the current market, splits the portfolio
into equal-weight allocations by basket, and logs a running TUNINDEX
snapshot that grows over time as you use it.

This is a screening and tracking tool, not a backtest. We do not have
access to full historical daily prices for Tunisian stocks, so instead
of inventing fake price history for real, named companies, the tool logs
one real snapshot per run and builds a genuine time series the more you
use it.

## Data sources

Three sites gave us usable, real data:

- ilboursa.com: the TUNINDEX level, the Tunisian Bond Index, Brent crude
  price, EUR/TND and USD/TND rates, the full list of companies on the
  exchange, and the day's top gainers and losers. Fetched 13 July 2026.
- tunisievaleurs.com: company background for Tunisie Valeurs, a
  brokerage founded in 1991.
- leconomistemaghrebin.com: a full article on Tunisie Valeurs' 2025
  portfolio recommendations, including the 16-stock list and Tunisia's
  H1 2024 macro numbers.

A fourth site, bmcecapitaltunisie.com, did not return readable content
through our fetch tool. No data from that site appears here. See
`data/README.md` and `report/sources.md` for the full breakdown of what
is official, what is generated by this tool, and what is missing.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

Check the model portfolio against the live market and print equal-weight
allocations:

```bash
python run_cross_reference.py
```

Add a new TUNINDEX snapshot on any day you check ilboursa.com:

```bash
python run_log_snapshot.py 2026-07-14 19900.50
```

Run the tests:

```bash
pytest tests/
```

## What is real and what is not

Every number in `data/` traces back to one of the three sites above and
carries a fetch date. The `sector_guess` column in the universe file is
our own rough label, not an official classification. The `portfolio_log.csv`
file starts with one real row and grows as you run the tracker, so later
rows depend on you entering real values by hand.
